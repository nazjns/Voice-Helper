## Chức năng
- Nhận diện giọng nói
- Chuyển giọng nói thành văn bản
- Hỗ trợ người dùng bằng AI
- Micro nhận diện giọng nói và hiển thị text realtime
- AI tóm tắt đoạn văn bản ghi được
- Hiển thị 3 - 5  key points từ nội dung
- Hỗ trợ tiếng Việt và tiếng Anh
- Tạo câu hỏi ôn tập dạng flashcard tự động (dành cho học sinh, sinh viên)
- Chế độ 'meeting mode': phân biệt nhiều người nói
- Tạo một màn hình "Lịch sử ghi chú" (Dashboard/History).
- Tích hợp lịch: gắn ghi chú vào ngày học cụ thể
- Xuất file .txt hoặc .md để lưu ghi chú
- Ô chat box "Hỏi trợ lý" (Q&A Chatbot) ngay bên cạnh màn hình ghi âm realtime.
